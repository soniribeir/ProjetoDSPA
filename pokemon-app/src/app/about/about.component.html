About PokemonApp:

This is an app developed as part of the Single Page Application Development module within the Java dot-Net course offered by the Upskill program.

The project consists of a frontend using Angular, which is an open-source web application framework.

By utilizing the services offered by the web API Softwium, users can explore a vast array of Pokémon-related content and functionalities.

From discovering detailed information about different Pokémon species to managing your own collection, this app aims to fulfill the diverse interests of Pokémon fans worldwide!
